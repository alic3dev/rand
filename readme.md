# rand

A simple utility program for generating various types of random values.

## Usage

```sh

USAGE: rand [options] [mode] [#length]
    Modes:
        hex     Hexadecimal
        dec     Decimal
        numalph Numeric + Alphabetical
        alph    Alphabetical
    Options:
        sec, --secure   Secure random generation
```

## Building

`make`
