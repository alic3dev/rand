# rand

generation->{values:random};

## usage

```zsh
usage: rand [options] [mode] [#length]
    modes:
        hex     hexadecimal
        dec     decimal
        numalph numeric|alphabetical
        alph    alphabetical
    options:
        sec, --secure   secure_random_generation
```

## build

```zsh
make
```
